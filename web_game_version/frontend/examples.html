Developer Examples

1. Flask API Usage

You can interact with the backend server directly using HTTP requests.

POST /predict HTTP/1.1 Host: localhost:5000 Content-Type: application/json { "image": "data:image/jpeg;base64,/9j/4AAQSk...", "category": "general" }

The server processes the image using MediaPipe and predicts the gesture.
